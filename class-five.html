Class Five

HTML 5 // CSS // Foundation 5 // jQuery

Class Five is a White Water Adventure guide and lodge for the advanced kayaker. This project included logo and branding for the Business as well as a website. Class Five offers paddlers different trip options as well as lodging or camping.

Responsive Website

Class Five’s website offers users the ability to sign-up and book trips, lodging, or campsites. This website is fully responsive, which means it can be viewed on a desktop, tablet, and smartphone.
