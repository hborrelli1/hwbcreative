Proteus Snowboards

WordPress // WooCommercer

A custom eCommerce solution for an innovative snowboard company that introduces the 1st "adjustable camber" snowboard. Built on WordPress with interactive javascript implementations showcasing the adjustable camber as well as a Build your own board customization option. After receiving the design for this project, the challenge was to create a custom tailored shopping experience for the user while also displaying animated graphics to show how the product works. The site design was produced by the awesome team at FlowState Marketing.

Visit Website

Customizing a board

I was challenged with creating a customized snowboard shopping experience where the user can customize the top layer, middle layer, and base layer of the board. The user can also customize the stiffness of the board. The solution was to utilize WooCommerce with custom plugins that allow the users to add custom options to the cart.

Displaying the layers

The next challenge was to create an interactive section showcasing each layer of the snowboard. The final solution involved converting the snowboard layers into svgs. JavaScript controls the transitions between the different layers and highlighting the corresponding sections. See it in action!
