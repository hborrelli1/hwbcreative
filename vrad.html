VRAD

React // React Testing Library

VRAD - Vacation Rentals Around Denver is a loose replicate of Airbnb. VRAD allows users to view different areas of Denver and view house listings available for rent. The user can favorite individual listings to refer back to at a later date. This was a paired project during my time at Turing School of Software & Design.

Learning Goals

For this project we took a TDD approach using React and React Testing Library. More learning goals for this project included writing squaky clean ES6 syntax, Think deeply about React Lifecycle Methods, Write tests for React components and some asynchronous functionality.
